What is the Document Object Model?

The Document Object Model (DOM) is a programming interface for web documents. It represents a webpage as a structured tree of objects, allowing developers to manipulate the content, structure, and styles dynamically using JavaScript or other programming languages.

What is the event delegation in the context of dom,and why it is useful?

Event delegation in the DOM is a technique where a single event listener is added to a parent element to manage events for multiple child elements. Instead of attaching separate listeners, the event propagates through the DOM, improving performance.

How do you manipulate an elements attribute and styles using the dom?

To manipulate an attribute in the DOM, you can use JavaScript's setAttribute(), getAttribute(), or removeAttribute() methods.
